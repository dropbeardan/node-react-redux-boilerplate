# Node React-Redux Boilerplate

Boilerplate code for a React app served by NodeJS and Webpack-Dev-Server.

## Stack

### Server
1. NodeJS (Server)
1. Express (API)
1. Axios (HTTP Request)
1. Jest (Test)
1. Cross-Env (Env Var)
1. Gulp (Task Runner)

### Client
1. React (Client DOM)
1. Redux (REACT Pattern)
1. LESS (CSS)
1. React/Redux Router (Routing)
1. Babel (Transpiler)
1. Webpack (Packager, Builder)
1. Jest (Test)
